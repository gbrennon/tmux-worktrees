# Switch between worktrees

Press `prefix` + `W` again.  The fzf picker lists every worktree you've created
so far — `feat/foo`, `fix/bar`, `chore/baz`, or whatever you named them.

Use the arrow keys (or keep typing to filter) and hit `Enter`.  You'll
jump straight to that worktree's tmux window.

Think of `prefix + W` as your "project dashboard" — one keystroke to see every
active branch and jump to any of them.
